# TO-DO-LIST

A simples to-do list application built with React and Typescript.

### Como rodar o projeto

1. Clone o repositório
2. Instale as depêndencias
  `npm i`
3. Execute o projeto
  `npm run dev`

### Tecnologias

- React
- TypeScript
- Vite

### Requisitos

- Input para cadastrar tarefas
- Botão para cadastrar tarefas
- Lista para exibir as tarefas pendentes
- Botão para deletar tarefas
- Botão para editar as tarefas
- Botão para concluir as tarefas
- Lista para exibir tarefas concluidas
